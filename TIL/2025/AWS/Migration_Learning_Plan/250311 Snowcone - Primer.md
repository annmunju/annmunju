- Snow Family 중 가장 작은 데이터 전송 디바이스 제품

### 시장의 과제
- 데이터를 클라우드에 올리거나
- 원격 위치에 통합하거나
- 열악한 환경에서 엣지 컴퓨팅을 수행
-> 오프라인으로 물리적 전송

### Snow Family
- 데이터 센터가 아닌 열악한 환경에서 운영해야하는 고객 지원
- AWS Snowcone, AWS Snowball Edge, AWS Snowmobile
- 물리적으로 엑사바이트 규모 데이터를 송수신

## Snowcone 소개

### 기능
1. 오프라인 및 온라인 데이터 전송
2. 컴퓨팅 용량
3. 프로세서 및 메모리
4. 연결된 스토리지
5. 네트워킹 기능
6. 소형 및 경량
7. 낮은 전력 요구 사항
8. 보안
9. 견고성
10. 확장된 운영 환경
11. 관리 및 운영
12. 내장 LCD 디스플레이

### 이점
- 보안
	- 하드웨어 계층에 대한 무단 액세스를 방지
	- 디바이스 액세스는 보호되며 잠금 해제 코드와 매니페스트 파일을 사용
- 휴대용 Edge 컴퓨팅
- 혹독한 환경 극복
- 유연한 데이터 전송 및 네트워킹
- IoT 센서 지원
- 일관된 환경: 클라우드에서 개발하고, Edge에서 컴퓨팅
