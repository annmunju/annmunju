### 분류 모델 
- 정확도 : 참 판별 / 전체 판별
- 정밀도 : 긍정적인 예측 중 맞는 예측 비율
	- 이메일 스팸 분류 모델. 스팸 분류를 잘못해 이메일 못보게 하는거 안됨
- 재현율 : 긍정적인 실제값 중 맞는 예측 비율
	- 불치병 예측 모델. 불치병이 없다고 예측하면 큰일
- AUC-ROC : AUC 면적이 1이 될수록 좋은 결과
