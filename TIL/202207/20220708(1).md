### 이미지 전처리

1. 이미지 변환
    
    1) 동차 좌표와 어파인 변환
    
    - 동차 좌표 : 2차원 점의 위치를 3차원 벡터로 표현. 마지막 원소로 1을 추가하여 만듦.
    - 어파인 변환 : 동차좌표 사용해 변환행렬을 곱하여 선형 변환을 하고 평행하게 이동하는 방법을 활용한 변환 `warpAffine`
    
    2) 강체 변환 = 유클리드 변환 : 회전(𝜃), 이동(𝑡)
    
    3) 유사 변환 : 확대/축소(s), 회전(𝜃), 이동(𝑡)
    
    `getRotationMatrix2D(center, angle, scale)`
    
    4) 3점 어파인 변환 `getAffineMatrix`
    
    5) 원근 변환 : 핀홀 카메라 모형을 사용
    
    - 사영행렬 계산 `getPerspectiveTransform`
    - 변환 함수 `warpPerspective`
    
2. 이미지 특징 추출
    
    1) 이미지 미분
    
    - 라플라스 연산을 통해 도함수의 크기 구하기
    - 중앙차분법을 사용해서 이미지 프로세싱 진행
    - 이미지 커널을 사용해 필터링
    
    2) 엣지 추출 : 경계선 인지
    
    - Sobel edge Detection `Sobel`
    , `Laplacian` 명령으로 구현
    
    3) 캐니 엣지 추출 : 그레디언트의 크기와 방향을 모두 사용하고 가짜 엣지를 제거해 엣지 구분 `canny`
    
    4) 코너 추출 : 엣지 교차점
    
    5) 해리스 코너 추출 : 이미지 위에 커널을 이동시키면서 변화량을 구하고 코너점을 찾음 `cornerHarris`
    
    6) Shi & Tomasi 코너 추출
    
    `goodFeaturesToTrack(image, maxCorners, qualityLevel, minDistance)`
    
    7) 허프 라인 변환 : 엣지 내 직선을 이루는 부분 찾기 `HoughLine`
    
    8) 이미지 히스토그램 : 이미지에서 특정 밝기 구간에 들어가는 픽셀의 수를 나타냄
    
    9) 그래디언트 히스토그램 설명자 : 특징점을 표현하는 벡터를 만들어 같은 특징점을 매칭하거나 추출 할 때 사용
    
    10) SIFT : 특징점 크기와 각도까지 같이 계산하여 이미지 크기가 변하거나 회전해도 동일한 특징점을 찾을 수 있도록 하는 방법
    
    11) SURF : 인텐서티 계산 방법을 간략화 하는 방법으로 sift의 속도와 안정성 개선
    
    12) FAST : 코너를 찾는 알고리즘. 빠른 연산으로 유명
    
    13) ORB : FAST 와 BRIEF를 기반으로 만들어진 알고리즘.
